## How to Run Locally

### วิธีที่ 1: XAMPP (Windows)
1. วางโฟลเดอร์ `k-mkt` ใน `C:\xampp\htdocs\`
2. เปิด XAMPP → Start Apache
3. เปิด Browser ไปที่ `http://localhost/k-mkt/`

### วิธีที่ 2: Live Server (VS Code)
1. ติดตั้ง Extension "Live Server"
2. คลิกขวาที่ `index.html` → Open with Live Server
3. เปิด `http://127.0.0.1:5500/`

### วิธีที่ 3: Python Simple Server
```bash
cd c:/xampp/htdocs/k-mkt
python -m http.server 8080
# เปิด http://localhost:8080
```
